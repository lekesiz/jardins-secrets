# Jardins Secrets — Site WordPress

Site web professionnel pour **Jardins Secrets**, entreprise paysagiste basée à Montbronn (Moselle).

## Projet Tuteuré — LPDWCA 2025/2026
**Groupe JS** : Mikail Lekesiz, Stéphane Vallejo, Virginie Sanfins, Catherine Braun

## Stack technique
- WordPress + Spectra One (thème parent)
- Spectra Gutenberg Blocks
- Hébergement : Hostinger
- Formulaires : Tally

## Structure du projet
- `style.css` — Child theme + CSS personnalisé (palette nature, responsive)
- `functions.php` — Fonctions du child theme (Gutenberg palette, Google Fonts, shortcodes)
- `templates/` — Templates de pages au format Gutenberg HTML
- `assets/js/` — JavaScript personnalisé (animations, FAQ accordion)

## Installation
1. Installer WordPress + Spectra One sur Hostinger
2. Copier `style.css` et `functions.php` dans `wp-content/themes/spectra-one-child/`
3. Activer le child theme
4. Créer les pages et copier le contenu HTML des templates
